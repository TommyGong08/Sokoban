## 配置汇编环境 

1. Visual Studio 2019配置32位汇编环境
2. 新建汇编空项目

如何配置汇编环境详见[本篇博客（保姆级教程）](https://blog.csdn.net/weixin_43794327/article/details/117320493?spm=1001.2014.3001.5501)  

## 正确放置项目文件  

分别将source,resource文件夹下的文件**复制粘贴**至VS2019对应的头文件、源文件和资源文件下。

## 如何操作
```
方向键上下左右控制小人移动  
        上
        ↑
  左←   ↓   →右
        下  

点击"重新开始"回到第一关  
点击"开始游戏"重新开始当前关卡  
```
